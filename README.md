Overview
This project is a JavaScript‑based recipe viewer application designed to strengthen my skills in client‑side JavaScript, DOM manipulation, ES6 array functions, recursion, and the use of external JavaScript libraries. The application displays recipes, allows filtering by ingredients, and dynamically generates recipe cards using JavaScript and CSS.

The purpose of this software is to demonstrate my ability to build interactive browser‑based applications using modern JavaScript techniques and to meet all module requirements for the JavaScript language module.
